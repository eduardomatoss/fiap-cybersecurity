# FIAP - CyberSecurity

The first CyberSecurity checkpoint consists of creating a solution with resources learned in the classroom in order to help identify possible terrorists inside airports, regardless of whether they are foreigners or residents of the same.

## Technology and Resources
- [Python](https://www.python.org/downloads/)
- [Google Colab](https://colab.research.google.com/)
- [pyngrok](https://pypi.org/project/pyngrok/)
- [streamlit](https://pypi.org/project/streamlit/)
- [streamlit-ace](https://pypi.org/project/streamlit-ace/)
- [fuzzywuzzy](https://pypi.org/project/fuzzywuzzy/)
